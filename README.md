# service-tevor
Simple ExpressJS API that connects to MongoDB via Mongoose

## Installation
Clone the repo
```
git clone url
```

Install dependencies
```
npm install
```

## Starting local dev 
Starting local dev server:
```
npm run dev
```

## Preparing for production build
Compile source code:
```
npm run build
```

Host in NodeJS:
```
npm run start
```

Alternatively, use ```node```:
```
node dist/app.js
```
